# megaline-revenue-analysis

# Descripción del Proyecto

Este proyecto consiste en un análisis estadístico de los datos de clientes de la empresa de telecomunicaciones Megaline. El objetivo principal es analizar el comportamiento de los usuarios de dos tarifas de prepago (Surf y Ultimate) para determinar cuál de ellas es más rentable. El análisis servirá como base para que el departamento comercial ajuste el presupuesto de publicidad de la compañía.

# Objetivos

* Procesamiento de datos: Limpiar y preparar datos de llamadas, SMS y navegación web.

* Análisis del comportamiento: Comparar el consumo promedio de minutos, mensajes y volumen de datos entre ambos planes.

* Cálculo de ingresos: Estimar los ingresos mensuales por usuario basándose en los cargos fijos y excedentes de cada plan.

* Prueba de hipótesis: Validar estadísticamente si existen diferencias significativas en los ingresos entre los planes y entre regiones geográficas (NY-NJ vs. otras ciudades).

# Tecnologías Usadas

* Lenguaje: Python 3.x

* Análisis de Datos: Pandas, NumPy

* Visualización: Matplotlib, Plotly

* Estadística: SciPy (para pruebas t de Student)
